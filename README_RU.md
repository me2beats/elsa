# ELSA - Агент эмпирического синтеза логики

ELSA (агент эмпирического синтеза логики) - экспериментальная система искусственного интеллекта, предназначенная для эмпирического синтеза логики. Планируется как интерактивная система решения текстовых задач и обучается в реальном времени.
